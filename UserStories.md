# User Stories
## Dungeon Master (DM)
As a DM I want to create questlines that can be hidden from my players.

As a DM I want to create individual bites of information for each quest that I can reveal to my players

As a DM I want to create rewards that can given out to my players when they complete certain quest steps.

As a DM I want to create secrets about a players character, that only them and I can see, so that we remember them for later

## Player
As a player I want to look up and search for quests, NPCs and items that my DM has revealed

As a player I want to personally comment on a quest, NPC or item to remember my thoughts about them
